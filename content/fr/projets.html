En voici quelques uns : * *[lldpd][1]* est un démon implémentant la norme [802.1AB][2]. Il permet d'envoyer des trames *LLDP* afin de notifier aux équipements connectés à la machine des informations telles que le nom de la machine, le nom du port, les VLAN présents, etc. Il gère également la réception ce qui autorise à déterminer que la machine est branchée sur tel port de tel équipement. Parmi les fonctionnalités intéressantes de ce démon, on notera le support de SNMP ainsi que des protocoles [CDP][3], SONMP et EDP. * *[wiremaps][4]* est une application web similaire à [NetDisco][5] dont le but principal est de répondre aux questions du type « quelle machine est branchée sur le port de ce switch » ou « sur quel switch se trouve cette machine ». Diverses informations sont collectées par SNMP telles que les bases LLDP, FDB ou ARP. * *[snimpy][6]* est un outil Python destiné à faciliter la création d'outils simples utilisant des requêtes SNMP. Son interface est très naturelle. Voici un example d'utilisation : #!python load("IP-FORWARD-MIB") m=M() print "Using IP-FORWARD-MIB::ipCidrRouteTable..." routes = m.ipCidrRouteNextHop for x in routes: net, netmask, tos, src = x print "%15s/%-15s via %-15s src %-15s" % (net, netmask, routes[x], src) * *[udpproxy][7]* permet de tunneler des requêtes *UDP* dans des tunnels SSH en utilisant la cible `NFQUEUE` de Netfilter. Il est ainsi possible de choisir quels sont les flux UDP à envoyer via le proxy.
